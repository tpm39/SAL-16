
To Run a Program
----------------

1) Ensure that the 1st line of the 'mc' file to be used has been deleted, if it was generated by the 'SAL-16' Assembler, and delete all the 'trailing zeros' from the 'mc' file.

2) In 'SAL_16.v' ensure the program to run is loaded into RAM by adjusting the following line to point to 'Prog':

   $readmemh("Prog.mc", RAM);

3) If any actions are required during a simulated program run, ensure that 'SAL_16_tb.v' has been updated accordingly.
